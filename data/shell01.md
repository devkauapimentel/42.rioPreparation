# Shell 01 — Advanced Shell
## Piscine C

---

### Chapter I: Instructions

* Shell scripts must be executable.
* All scripts tested on standard bash.

---

### Chapter II: Mandatory Part

#### Exercise 00: print_groups
* **Turn-in directory:** `ex00/`
* **Files to turn in:** `print_groups.sh`

Write a command that displays the list of groups the current user belongs to, separated by commas without trailing spaces.

---

#### Exercise 01: find_sh
* **Turn-in directory:** `ex01/`
* **Files to turn in:** `find_sh.sh`

Write a command that searches for all files ending in `.sh` in the current directory and subdirectories. Display only the file names without the `.sh` extension. One per line.

---

#### Exercise 02: count_files
* **Turn-in directory:** `ex02/`
* **Files to turn in:** `count_files.sh`

Write a command that counts the total number of regular files and directories (recursively) in the current directory. The current directory itself is not counted.

---

#### Exercise 03: find_sh_2
* **Turn-in directory:** `ex03/`
* **Files to turn in:** `find_sh_2.sh`

Write a command that searches for all `.sh` files in the current directory and subdirectories, then modifies their permissions to add execute for user, read for group, and nothing for others.

---

#### Exercise 04: MAC
* **Turn-in directory:** `ex04/`
* **Files to turn in:** `mac.sh`

Write a command that displays the MAC addresses of all network interfaces, one per line.
