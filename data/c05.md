### Chapter II: Mandatory Part

#### Exercise 00: ft_iterative_factorial
* **Turn-in directory:** `ex00/`
* **Files to turn in:** `ft_iterative_factorial.c`
* **Allowed functions:** None

Create an **iterative** function that returns the factorial of the parameter. If the argument is not valid, the function should return `0`. Overflows are not handled.
```c
int ft_iterative_factorial(int nb);
```

* **Turn-in directory:** `ex02/`

#### Exercise 04: ft_fibonacci
* **Turn-in directory:** `ex04/`
* **Files to turn in:** `ft_fibonacci.c`
* **Allowed functions:** None

Create a function that returns the `n`th element of the Fibonacci sequence, the first element being at index `0`. Return `-1` if the index is negative.
```c
int ft_fibonacci(int index);
```

> **Trap:** `fib(0) = 0`, `fib(1) = 1`, `fib(n) = fib(n-1) + fib(n-2)`.
