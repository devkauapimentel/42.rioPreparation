### Chapter II: Mandatory Part

#### Exercise 00: ft_ft
* **Turn-in directory:** `ex00/`
* **Files to turn in:** `ft_ft.c`
* **Allowed functions:** None

Create a function that takes a pointer to `int` as a parameter, and sets the value `42` to that `int`.
```c
void ft_ft(int *nbr);
```

> **Trap:** `*nbr = 42;` NOT `nbr = 42;`. Dereferencing is the key concept here.

#### Exercise 01: ft_ultimate_ft
* **Turn-in directory:** `ex01/`
* **Files to turn in:** `ft_ultimate_ft.c`
* **Allowed functions:** None

Create a function that takes a pointer-to-pointer-to-pointer-to-pointer-to-pointer-to-pointer-to-pointer-to-pointer-to-pointer to `int` (9 levels of pointers) and sets the value `42` to that `int`.
```c
void ft_ultimate_ft(int *********nbr);
```

> **Trap:** `*********nbr = 42;` — nine dereferences. Count them carefully.

#### Exercise 02: ft_swap
* **Turn-in directory:** `ex02/`
* **Files to turn in:** `ft_swap.c`
* **Allowed functions:** None

Create a function that swaps the value of two integers whose addresses are entered as parameters.
```c
void ft_swap(int *a, int *b);
```

> **Trap:** You need a temporary variable. `int tmp = *a; *a = *b; *b = tmp;`

#### Exercise 04: ft_ultimate_div_mod
* **Turn-in directory:** `ex04/`
* **Files to turn in:** `ft_ultimate_div_mod.c`
* **Allowed functions:** None

Create a function that divides the `int` pointed to by `a` by the `int` pointed to by `b`. The result of the division is stored in the `int` pointed to by `a`. The remainder is stored in the `int` pointed to by `b`.

#### Exercise 05: ft_putstr

Create a function that displays a string of characters on the standard output.
```c
void ft_putstr(char *str);
```

> **Trap:** Iterate until `*str == '\0'`. Use `write(1, str, 1); str++;`.

#### Exercise 07: ft_rev_int_tab
* **Turn-in directory:** `ex07/`
* **Files to turn in:** `ft_rev_int_tab.c`
* **Allowed functions:** None

Create a function that reverses a given array of integers. `size` is the number of elements in the array.
```c
void ft_rev_int_tab(int *tab, int size);
```

> **Trap:** Swap from both ends toward the center. `tab[i]` ↔ `tab[size - 1 - i]`.
