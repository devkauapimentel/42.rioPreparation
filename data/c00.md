# C00 — Introduction to C
## Piscine C

---

### Chapter I: Instructions

* Only the function `write` is allowed (from `<unistd.h>`).
* Compilation must succeed with `gcc -Wall -Wextra -Werror`.
* The Norm must be respected at all times.
* No `for`, `switch`, or `do...while` loops.
* Max 25 lines per function. Max 5 variables per function.
* Your turn-in directory must be named exactly as specified.

---

### Chapter II: Mandatory Part

#### Exercise 00: ft_putchar
* **Turn-in directory:** `ex00/`
* **Files to turn in:** `ft_putchar.c`
* **Allowed functions:** `write`

Write a function that displays the character passed as a parameter.
It will be prototyped as follows:
```c
void ft_putchar(char c);
```

> **Hint:** The `write` function takes a file descriptor, a pointer to a buffer, and a byte count: `write(1, &c, 1);`

#### Exercise 01: ft_print_alphabet
* **Turn-in directory:** `ex01/`
* **Files to turn in:** `ft_print_alphabet.c`
* **Allowed functions:** `write`

Write a function that displays the alphabet in lowercase, on a single line, by ascending order, starting from the letter `'a'`.
```c
void ft_print_alphabet(void);
```

> **Trap:** How do you loop without `for`? A `char` is just a number. `char c = 'a'; while (c <= 'z')`.

#### Exercise 05: ft_print_comb
* **Turn-in directory:** `ex05/`
* **Files to turn in:** `ft_print_comb.c`
* **Allowed functions:** `write`

Write a function that displays all different combinations of three different digits in ascending order, separated by a comma and a space.

Expected output begins with: `012, 013, 014, 015, ...` and ends with `789`.

```c
void ft_print_comb(void);
```

> **Trap:** Three nested `while` loops. Each inner counter starts one above the outer. `i < j < k`.

#### Exercise 06: ft_print_comb2
* **Turn-in directory:** `ex06/`
* **Files to turn in:** `ft_print_comb2.c`
* **Allowed functions:** `write`

Write a function that displays all different combinations of two two-digit numbers (from `00` to `99`), separated by a comma and a space. The first number must be smaller than the second.

Expected: `00 01, 00 02, ..., 98 99`.
```c
void ft_print_comb2(void);
```

> **Trap:** You need to display leading zeros. To display `05`, write `'0'` then `'5'` separately.

#### Exercise 07: ft_putnbr
* **Turn-in directory:** `ex07/`
* **Files to turn in:** `ft_putnbr.c`
* **Allowed functions:** `write`

Write a function that displays the number passed as a parameter. The function must handle all valid `int` values.

#### Exercise 08: ft_print_combn
* **Turn-in directory:** `ex08/`
* **Files to turn in:** `ft_print_combn.c`
* **Allowed functions:** `write`

Write a function that displays all different combinations of `n` different digits in ascending order.

`n` will be between 1 and 9 inclusive.
```c
void ft_print_combn(int n);
```

> **Trap:** Recursion or variable-depth nested loops. This is the hardest exercise in C00.
