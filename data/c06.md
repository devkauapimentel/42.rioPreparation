# C06 — Command-line Arguments
## Piscine C

---

### Chapter I: Instructions

* Only `write` is allowed unless stated otherwise.
* Compilation: `gcc -Wall -Wextra -Werror`.
* The Norm must be respected.

---

### Chapter II: Mandatory Part

#### Exercise 00: ft_print_program_name
* **Turn-in directory:** `ex00/`
* **Files to turn in:** `ft_print_program_name.c`
* **Allowed functions:** `write`

Create a program that displays its own name followed by a newline.

---

#### Exercise 01: ft_print_params
* **Turn-in directory:** `ex01/`
* **Files to turn in:** `ft_print_params.c`
* **Allowed functions:** `write`

Create a program that displays its arguments, one per line. Program name excluded.

---

#### Exercise 02: ft_rev_params
* **Turn-in directory:** `ex02/`
* **Files to turn in:** `ft_rev_params.c`
* **Allowed functions:** `write`

Display arguments in reverse order, one per line.

---

#### Exercise 03: ft_sort_params
* **Turn-in directory:** `ex03/`
* **Files to turn in:** `ft_sort_params.c`
* **Allowed functions:** `write`

Display arguments sorted in ASCII order, one per line.
